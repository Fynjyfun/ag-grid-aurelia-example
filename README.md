
ag-Grid
==============

Example of using ag-Grid with Aurelia and TypeScript

Aurelia CLI
==============
http://aurelia.io/hub.html#/doc/article/aurelia/framework/latest/the-aurelia-cli

Aurelia CLI should be installed globally. 
- 'npm install aurelia-cli -g'


Building
==============

To build:
- `npm install`
- `au build`

To Run:
- `au run --watch`
